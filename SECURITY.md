# Security Policy

## Supported Versions

| Version | Supported |
|---------|----------|
| 0.1.x   | ✅        |

## Reporting a Vulnerability

Email **jmsernatuya@gmail.com** with subject: `[SECURITY] latamdata-py`

Do **not** open public GitHub issues for security vulnerabilities.
Response within 48 hours; full response within 7 days.

## Scope

- Code execution vulnerabilities in the package
- Dependency vulnerabilities (supply chain)
- Data integrity issues that could corrupt research outputs
